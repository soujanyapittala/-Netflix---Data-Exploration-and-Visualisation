# -Netflix---Data-Exploration-and-Visualisation
Netflix Content Strategy Analysis using EDA and visualization to uncover trends in movies vs TV shows, genres, countries, and release patterns. Focused on deriving actionable insights to guide content production, market expansion, and data-driven decision-making.
🎯 Objectives
Compare Movies vs TV Shows growth trends
Identify high-performing genres
Analyze country-wise content distribution
Understand release timing patterns
Generate data-driven business recommendations
📂 Dataset
Source: Netflix Movies & TV Shows dataset
Features: Title, Type, Director, Cast, Country, Release Year, Rating, Duration, Genre
🧠 Approach
Data Cleaning (missing values, duplicates)
Data Transformation (unnesting cast, country, genres)
Exploratory Data Analysis (EDA)
Visualization & Trend Analysis
📊 Key Insights

👉 TV Shows have grown significantly in recent years, indicating a strategic shift from Movies

👉 Content is heavily concentrated in a few countries, highlighting expansion opportunities

👉 Certain genres dominate consistently, suggesting stable audience demand

👉 Content additions have increased post-2015, reflecting aggressive scaling

💡 Recommendations
Increase investment in TV Shows for long-term engagement
Expand production in underrepresented regions
Focus on high-performing genres for better ROI
Optimize release timing based on observed trends
🛠️ Tech Stack
Python (Pandas, NumPy)
Matplotlib, Seaborn
Jupyter Notebook
